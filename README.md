# AtarImage — Web AR (MindAR.js)
Free image-tracking AR. No account. No API key. Works on any phone via GitHub Pages.

---

## 📁 Files in this folder

```
AtarImage/
├── index.html            ← Main AR page (upload to GitHub)
├── compile-target.html   ← Run this LOCALLY first (do NOT upload)
├── assets/
│   ├── imgtracker.png    ← Copy from your Desktop project
│   └── targets.mind      ← Generated by compile-target.html
└── README.md
```

---

## ⚡ Setup (3 steps)

### Step 1 — Copy your image
Copy `imgtracker.png` into the `assets/` folder:
```
From: D:\8thWall\Atar\AtarImage\src\assets\imgtracker.png
To:   assets\imgtracker.png
```

### Step 2 — Generate targets.mind
1. Open `compile-target.html` in Chrome (just double-click it)
2. Drop `imgtracker.png` onto the page
3. Click **Compile** — it will auto-download `targets.mind`
4. Move `targets.mind` into the `assets/` folder

### Step 3 — Upload to GitHub & enable Pages
Upload this entire folder to a GitHub repo, then:
- **Settings → Pages → Branch: main → / (root) → Save**

Your live AR link:
```
https://helmiedt.github.io/AtarImage/
```
